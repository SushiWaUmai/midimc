## Getting Started

### Prerequisites

- node.js
- pnpm (optional)

### Run Locally

Clone the repo

```bash
git clone https://github.com/SushiWaUmai/midimc.git
cd midimc
```

Install dependencies and run the development server

```bash
pnpm install # or npm if you don't have pnpm
pnpm dev
```
